Для создания ссылок используется тег a, с атрибутом href, указывающих на адрес: Ссылка на Google.

Внутри ссылок может располагаться не только текст, но и любой другой блок HTML:

Другая ссылка на Google

Ссылку можно открыть в новой вкладке. Для этого в атрибут target необходимо поместить значение _blank: Третья ссылка на Google, открывающаяся в новой вкладке

Помимо ссылок на другие страницы в интернете, можно ссылаться на страницы у нас. Страница посвященная стилям Страница посвященная секциям
