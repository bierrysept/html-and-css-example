Про CSS

Вступление

Что такое CSS?

  • CSS расшифровывается, как Cascading Style Sheets, то есть таблицы рекурсивной стилизации
  • CSS объясняет, как HTML елементы должны отображаться на экране, бумаге или где бы то ни было ещё.
  • CSS экономит много времени. Он позволяет контролировать отображение множества страниц с одним стилем
  • Внешние таблицы стилей сохраняются в CSS-файлах

Для чего использовать CSS

CSS используется для определения стилей страниц, включая дизайн, макет и вариации для разных устройств и экранов

CSS решает большую проблему

HTML никогда не был предназначен для форматирования сайтов. Он был разработан для разметки страниц. Когда тег font и цветовые атрибуты были добавлены в HTML 3.2 это превратилость в кошмар для разработчиков. Разработка больших сайтов, у которых информация о шрифтах и цветах была добавлена на каждую страницу, превратилась в долгий и дорогой процесс.

Чтобы решить эту проблему W3C создало CSS.

CSS удалил деклорацию стилей из HTML-страниц!

CSS спасает много работы

Определение стилей отлично сохраняются в *.css файлах.

C внешними файлами можно изменять стиль всего сайта, изменяя всего лишь 1 *.css файл!

Синтаксис CSS

Правила CSS состоят из селектора(selector) и блока деклараций (declaration block).

Синтаксис CSS

h1 {color:blue; font-size:12px}

Селектор определяет множество элементов, которые мы хотим стилизовать

Блок деклараций содержит одину или несколько деклараций, разделенных точкой с запятой

Каждая декларация содержит имя и значение свойства CSS, разделеннные двоеточием.

Множество деклараций, разделенных ";", образуют блоки декларации, которые окружены фигурными скобками.

В этом примере все теги p будут отцентрированы, а текст будет перекрашен в красный:

p {
  color: red;
  text-align: center;
}

Объяснение примера

  • p - это селектор в CSS(который указывает на элемент, что нужно стилизовать: на <p>)
  • color - это свойство, а red - то значение свойства
  • text-align - это свойство, а center - то значение свойства

Селекторы CSS

Селекторы выбирают элемент(ы), которые вы хотите стилизировать.

Селекторы CSS

Селекторы используются чтобы "искать" (или выбирать) элементы HTML, которые вы хотите стилизировать.

Мы можем разделить селекторы на 5 категорий:

  • Простые селекторы (выбирают элементы на основании имени, id или класса)
  • Комбинаторные селекторы (выбирают элементы, основные на специфичных взаимоотношениях между ними)
  • Селекторы псевдокласса (выбирают элементы на основании их определенного состояния)
  • Селекторы псевдоэлемента (выбирают и стилизуют часть элемента)
  • Селекторы по аттрибутам (Выбирает элементы на основании аттрибутов или их значений)

Сейчас будут объяснены основные виды селекторов.

Селектор элемента CSS

Селектор элементов выбирает HTML элементы на основании имяни элемента 

p {
  text-align: center;
  color: red;
}

Таким образом, будут найдены все теги p.

Селектор по id

Селектор по id использует аттрибут элемента id чтобы выбрать конкретный элемент.

Id на странице может быть 1, так что селектор выберет только 1 элемент.

Для выбора элемента по специфичному id, напишите решётку (#) прямо перед названием id элемента 

#para1 {
  text-align: center;
  color: red;
}

Таким образом, будет выбран элемент у которого указан id="para1"

Кстати, id не может начинаться с числа.

Селектор класса

Селектор класса выбирает элементы HTML с специфичксуим аттрибутом.

Для выбора элементов с определенным классом напишите точку (.) прямо перед именем класса 

.center {
  text-align: center;
  color: red;
}

В этом примере все HTML элементы с class="center" будут окрашены в красный и отцентрованы.

Также можно указать, какие конкретно теги HTML должны быть выбраны с классом. 

p.center {
  text-align: center;
  color: red;
}

В этом примере только элементы <p> с class="center" будут окрашены в красный и отцентрованы.

Также HTML-элементы могут отсылаться на больше чем 1 класс. 

<p class="center large">This paragraph refers to two classes.</p>

Кстати, класс не может начинаться с числа.

Всеобщий селектор

Всеобщий селектор (*) выбирает все элементы на странице 

* {
  text-align: center;
  color: blue;
}

Это CSS правило будет распространяться на все элементы страницы.

Группировка селекторов

Группировка селекторов выбирает все элементы HTML с одним и тем же определением стилизации.

Посмотрите на этот код: 

h1 {
  text-align: center;
  color: red;
}

h2 {
  text-align: center;
  color: red;
}

p {
  text-align: center;
  color: red;
}

Тут h1, h2 и p элементы имеют одно определение.

Будет лучше сгруппировать селекторы для минимизации кода.

Для группировки селекторов, разделите их запятыми. 

h1, h2, p {
  text-align: center;
  color: red;
}
