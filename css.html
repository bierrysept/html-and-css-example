Про CSS

Вступление

Что такое CSS?

  • CSS расшифровывается, как Cascading Style Sheets, то есть таблицы рекурсивной стилизации
  • CSS объясняет, как HTML елементы должны отображаться на экране, бумаге или где бы то ни было ещё.
  • CSS экономит много времени. Он позволяет контролировать отображение множества страниц с одним стилем
  • Внешние таблицы стилей сохраняются в CSS-файлах

Для чего использовать CSS

CSS используется для определения стилей страниц, включая дизайн, макет и вариации для разных устройств и экранов

CSS решает большую проблему

HTML никогда не был предназначен для форматирования сайтов. Он был разработан для разметки страниц. Когда тег font и цветовые атрибуты были добавлены в HTML 3.2 это превратилость в кошмар для разработчиков. Разработка больших сайтов, у которых информация о шрифтах и цветах была добавлена на каждую страницу, превратилась в долгий и дорогой процесс.

Чтобы решить эту проблему W3C создало CSS.

CSS удалил деклорацию стилей из HTML-страниц!

CSS спасает много работы

Определение стилей отлично сохраняются в *.css файлах.

C внешними файлами можно изменять стиль всего сайта, изменяя всего лишь 1 *.css файл!
